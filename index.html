Install 4chan X

Using Opera? Install Opera 12+ (currently in beta). Why?
Chrome:
  1. Install 4chan X, that's it.
Firefox:
  1. Install Scriptish.
  2. Restart Firefox.
  3. Install 4chan X.
Opera:
  1. Go to Preferences (Ctrl + F12) → Advanced → Content → JavaScript Options → User JavaScript Folder.
  2. Chose what will be your userscripts folder.
  3. Save the 4chan X file into your userscripts folder.
Opera Mobile:
  1. Enter opera:config in the address bar.
  2. Under User Prefs, enable User JavaScript.
  3. Set User JavaScript File to what will be your userscripts folder, /sdcard/js for example
  4. Save your settings.
  5. Save the 4chan X file into your userscripts folder.

Known issues:
  • Clearing your 4chan cookies will erase your settings!
Chrome:
  • 4chan X only runs on 4chan! Chrome will say that it runs on all domains due to its match patterns.
Firefox:
  • If you get an error stating localStorage is null, go to about:config and enable dom.storage.enabled.
  • If you get a Security error, make sure cookies are enabled.
  • Linkify Plus: add // @delay 1 to its metadata block.
  • NoScript: for version updating to work, you must allow github.com.
  • Scriptish: for Scriptish to version update 4chan X, disable Scriptish's Require user scripts to update using HTTPS preference: Scriptish Preferences > Advanced > Update Security.

If your issue is not in the list above, you may have found a new bug. Before reporting it...

  1. Make sure your browser & 4chan X are up to date.
    • Legacy and unstable browsers are not supported; if you find an issue with them, report it to their bug trackers: Firefox, Chrome.
  2. Disable your other extensions & scripts.

If your issue persists, report precise steps to reproduce, and any console errors. Open your console with:

  • Ctrl + Shift + J on Chrome
  • Ctrl + Shift + K on Firefox
  • Ctrl + Shift + O on Opera.

DownThemAll for Firefox: download all images in a thread with this fast filter: /^http://images.4chan.org/.
*nix: Download all images in a thread with this shell function (add it to your .bashrc): 

dta() {
  wget -O - $1 |
  grep -Eo 'images.4chan.org/[^"]+' |
  uniq |
  xargs wget
}
