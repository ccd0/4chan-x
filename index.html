A joint project of Zixaphir and Seaweed.

The new v3-based version is out!

Make sure you export your settings and disable your old userscript before installing.

The latest stable releases of Chrome, Firefox, and Opera are supported.

Install 4chan X

Chrome:
  1. Install 4chan X. That's it.
  2. 4chan X will be automatically updated by Chrome.
Firefox:
  1. If you don't have Greasemonkey installed:
    1. Install Greasemonkey.
    2. Restart Firefox.
  2. Set layout.css.flexbox.enabled to true in your about:config.
  3. Install 4chan X.
Opera:
  1. If you haven't set your userscripts folder:
    1. Go to Preferences (Ctrl + F12) → Advanced → Content → JavaScript Options → User JavaScript Folder.
    2. Chose what will be your userscripts folder.
    3. Enable User JavaScript on HTTPS and save your settings.
    4. Set User JS Storage Quota to a minimum of 5120.
  2. Save the 4chan X file into your userscripts folder.
